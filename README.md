# Quantity Measurement App – Frontend

A modern and responsive frontend application built using **Angular** for the Quantity Measurement App.
This frontend communicates with backend microservices through the API Gateway and provides users with a smooth and secure experience.

The project demonstrates real-world frontend development concepts such as **Angular Components**, **Routing**, **Reactive Forms**, **JWT Authentication**, **REST API Integration**, and responsive UI design.

---

## 🚀 Tech Stack

* Angular
* TypeScript
* HTML5
* CSS3 / SCSS
* Bootstrap / Tailwind CSS
* Angular Router
* Angular Reactive Forms
* RxJS
* REST APIs
* JWT Authentication
* Vercel Deployment Ready

---

## 📌 Application Flow

```text id="fw6g12"
User
  |
  v
Angular Frontend
  |
  v
API Gateway
  |
  v
Backend Microservices
(Auth Service + QMA Service)
```

The frontend interacts with backend services through a centralized API Gateway, ensuring clean architecture and secure communication.
